# Development Environment Setting Repository

대충 개발환경 세팅 자동화하려는 레포

## 사용법
1. 이 레포를 clone한다.
```sh
git clone https://github.com/mozylee/dev-env-setting.git
```

2. 클론한 위치로 이동하여 아래의 명령어를 실행
```sh
sh install.sh
```
